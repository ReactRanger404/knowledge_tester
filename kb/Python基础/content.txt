Python 是一种高级编程语言，它的设计哲学强调代码的可读性和简洁性。

Python 的基本数据类型包括：
1. 整数（int）：表示整数，如 42、-3、1000
2. 浮点数（float）：表示小数，如 3.14、-0.5、2.0
3. 字符串（str）：表示文本，如 "hello"、'Python'、"123"
4. 布尔值（bool）：表示 True 或 False

Python 的容器类型：
- 列表（list）：有序可变序列，用方括号 [] 定义。如 [1, 2, 3]
- 元组（tuple）：有序不可变序列，用小括号 () 定义。如 (1, 2, 3)
- 字典（dict）：键值对集合，用花括号 {} 定义。如 {"name": "Tom", "age": 18}
- 集合（set）：无序不重复元素，用 set() 或 {} 定义。如 {1, 2, 3}

流程控制：
- if/elif/else：条件判断
- for 循环：遍历可迭代对象
- while 循环：条件循环

函数的定义使用 def 关键字。
类使用 class 关键字定义。
面向对象三大特性：封装、继承、多态。
